Expressions faciales

Jette un œil à ces expressions faciales et essaie-les dans tes propres dessins.

Pour donner l'impression que ton personnage d'anime est amoureux, remplace les yeux par deux cœurs arrondis. Tu peux ajouter trois cœurs supplémentaires à l'intérieur pour un effet amusant.

Une expression heureuse en anime utilise deux lignes courbes épaisses, pour les yeux et une bouche large pour créer un visage heureux et rire.

Une expression faciale ennuyée peut être faite en pointant les sourcils vers le centre. Les yeux utilisent une ligne épaisse avec un demi-cercle en dessous. La bouche est légèrement courbée.
