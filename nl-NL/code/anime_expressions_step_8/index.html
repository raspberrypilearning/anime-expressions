Om je anime-personage eruit te laten zien alsof hij of zij verliefd is, vervang je de ogen door twee ronde harten. Je kunt nog drie harten toevoegen voor een leuk effect.

Een Vrolijke uitdrukking in anime gebruikt twee dikke gebogen lijnen voor de ogen en een brede mond om een blij, lachend gezicht te creëren.

Een geïrriteerde gezichtsuitdrukking kan worden gemaakt door de wenkbrauwen naar het midden te richten. Voor de ogen wordt een dikke lijn met een halve cirkel eronder gebruikt. De mond is licht gebogen.
